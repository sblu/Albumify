# Albumify

Train and deploy a tiny on-device model that turns an album cover into a black-and-white line drawing. Runs on a Raspberry Pi 5 with ~5 MB on disk and ~200 MB RAM.

See `docs/superpowers/specs/2026-05-20-albumify-design.md` for the full design and `docs/superpowers/plans/2026-05-20-albumify.md` for the implementation plan.
